# monte-cinema

Create database:

`$ bin/rails db:create`

Drop database:

`$ bin/rails db:drop`

Start server:

`$ bin/rails server`

Generate controller:

`$ bin/rails generate controller CONTROLLER_NAME --skip-routes`

Generate model:

`$ bin/rails generate model MODEL_NAME COLUMN_NAME:COLUMN_TYPE...`

Run migration:

`$ bin/rails db:migrate`

Open console:

`$ bin/rails console`
